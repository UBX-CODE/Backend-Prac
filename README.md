# Backend Practice Repository

Welcome to my Backend Practice repository! This is a monorepo containing various backend projects, architectures, and experiments to build and practice modern backend development skills.

## 🚀 Projects Included

1. **Auth-system**: A complete authentication and authorization system.
2. **Chat-App**: A real-time chatting application.
3. **Event-hub**: An event-driven architecture setup or event management system.
4. **Logger-system**: A centralized logging service or custom logger implementation.
5. **Rate-limiter**: A rate limiting service or middleware to protect APIs from abuse.
6. **Resume-parser**: A system to parse and extract information from resumes.
7. **url-shortener**: A URL shortening service.

## 🛠️ Tech Stack

This repository encompasses multiple stacks and technologies tailored to each specific project's requirements.

## 📦 Getting Started

Each project is contained within its own directory. To run a specific project, navigate to its directory, install the required dependencies, and start the development server.

```bash
# Example
cd Chat-App
npm install
npm start
```
